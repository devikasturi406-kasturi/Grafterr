** Grafterr Website Project **

This is a modern landing page I built for the Grafterr tech challenge. It’s designed to look professional, and handle data smoothly.

🔗 See it Live
https://grafterr.onrender.com/

🎨 What I Built

1. Error Handling: If the internet fails, a "Try Again" button appears so the user isn't stuck.

2. Carousel Slider: A smooth product slider that shows 3 items on a computer, 2 on a tablet, and 1 on a phone.

3. Easy Updates: All the text and images come from a single file (content.json). If you change the text there, the whole website updates instantly.

📂 How it's Organized

1. index.html: The main structure of the page.

2. css/: All the "makeup" (colors, fonts, and layout).

3. js/: The "brain" of the site (fetching data and making the slider move).

4. data/: The file that holds all the website's text.

5. images/: The logos and pictures.

🛠️ How to run it on your computer

1. Download the files.

2. Because this uses modern JavaScript "Modules," you cannot just double-click the file.

3. You need to use a tool like "Live Server" in VS Code to see it work properly.
